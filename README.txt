使用说明：
1.从视频网站下载FLV，命名为hhj+数字.flv。数字从1开始，数字要连续。然后放在Sys文件夹下。
2.修改Sys文件夹下的hhj.ini。将flvnum改为视频的个数。
3.修改Sys文件夹下的hhj.ini。将btnname1、btnname2、btnname3分别改为程序目录下的3个文件夹，具体自己安排。比如Video，Anime，Pics，Games等等。注意要用英文。貌似由于按钮宽度限制，不能超过8个字母。这三个变量对应自启动程序中的三个按钮。

技术说明：
1.按钮上的文字的字体是Tahoma。
2.为了美观，建议btnname系列的第一个字母大写。
3.视频缓冲时间为5秒。
